Mi Página Web

Hola, bienvenidos a mi página web. Aquí les voy a mostrar algunos ejemplos de cosas que podemos hacer con HTML.

Por ejemplo, cuando hablamos de HTML, estamos hablando de un lenguaje que usamos para crear páginas web.

La palabra HTML significa "HyperText Markup Language", o sea, "Lenguaje de Marcado de Hipertexto". Es el lenguaje básico de la web.

"La web es una herramienta para cambiar el mundo" – Tim Berners-Lee.
